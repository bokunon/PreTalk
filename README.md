# PreTalk

初回商談の下調べをまとめるツールです。

## 概要

会社名・部署名・担当者名を入力すると、IR情報・理念・ビジョン・中期経営計画・インタビュー記事・ニュースなどを自動で収集し、初回商談で話すトピック・提案ネタをまとめます。

## 使い方

1. 会社名・部署名（必須）・担当者名（任意）を入力
2. 「下調べを実行」をクリック
3. 1〜2分待つと、会社概要・部署の文脈・課題推測・提案ネタ・初回商談トピックが表示されます

## セットアップ

```bash
# 依存関係のインストール
npm install

# 環境変数の設定
cp .env.example .env
# .env に OPENAI_API_KEY を設定

# 開発サーバー起動
npm run dev
```

http://localhost:3000 でアクセスできます。

## 技術スタック

- Next.js (App Router)
- DuckDuckGo 検索（APIキー不要）
- OpenAI API（分析・統合）
